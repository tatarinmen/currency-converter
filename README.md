## 📝 Описание
Графическое приложение на Python для конвертации валют с использованием актуальных курсов из внешнего API. Поддерживает выбор валют из выпадающего списка, мгновенную конвертацию, сохранение истории операций в формате JSON и автоматическую загрузку данных при запуске.

### ✨ Возможности:
- 🌍 Поддержка 160+ валют мира через ExchangeRate-API
- 💱 Мгновенная конвертация с расчётом по актуальному курсу
- 📊 Таблица истории с деталями: время, валюты, сумма, результат, курс
- 💾 Автосохранение/загрузка истории в `conversion_history.json`
- 🔄 Кнопка обновления курсов валют
- ✅ Валидация ввода: только положительные числа
